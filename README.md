Welcome to My Portfolio!

This is my personal portfolio created using Node.js, Express, and EJS. It showcases my experience, as well as a bit about my hobbies and projects. 

The page has four pages- Home, About Me, Projects, and Contact. I have used Bootstrap 5 for layout and responcivenss, and Font Awesome for icons. Used some of my favrite colours for linear gradients buttons. And i like checkers, so have a personal aesthic background. The code is fully styles wuth Bootstrap classes.
